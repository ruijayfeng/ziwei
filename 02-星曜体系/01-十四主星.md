# 十四主星详解

## 概述

十四主星是紫微斗数的核心星曜，分为两大星系：

| 星系 | 星曜 | 排布方向 |
|------|------|---------|
| 紫微星系 | 紫微、天机、太阳、武曲、天同、廉贞 | 逆时针 |
| 天府星系 | 天府、太阴、贪狼、巨门、天相、天梁、七杀、破军 | 顺时针 |

### 四大类型划分

| 类型 | 星曜 | 特质 |
|------|------|------|
| 领导型 | 紫微、天府、武曲、天相 | 稳重权威、决策力强 |
| 开创型 | 七杀、破军、廉贞、贪狼 | 冲劲锐气、开拓进取 |
| 支援型 | 太阳、巨门、天机 | 智谋辅助、善于表达 |
| 合作型 | 太阴、天梁、天同 | 温和协调、善于配合 |

---

## 一、紫微星系（6颗）

### 1. 紫微星（帝王星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴土 |
| 斗系 | 北斗主星 |
| 化气 | 尊贵 |
| 代表人物 | 周文王 |

**性格特质：**
- **优点**：领导力强、高贵威严、仁慈宽厚、有决断力
- **缺点**：主观固执、好面子、孤傲、依赖性强

**关键特点：**
- 帝王之星，需要辅佐星（左辅右弼）才能发挥最大能量
- 独坐命宫缺乏吉曜 → 决策过于主观
- 有天府、左右相伴 → 一呼百诺

**受亮度影响**：❌ 不受影响

---

### 2. 天机星（智囊星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴木 |
| 斗系 | 南斗 |
| 化气 | 善、兄弟主 |
| 代表人物 | 姜子牙 |

**性格特质：**
- **优点**：聪明机智、思维敏捷、学习力强、善于策划
- **缺点**：多思多虑、神经质、容易焦虑、投机取巧

**关键特点：**
- 军师谋臣之星，适合辅佐他人
- 不宜独立创业，适合幕僚角色
- 善变、喜欢新鲜事物

**受亮度影响**：✅ 受影响（庙旺聪明善良，落陷多疑善变）

---

### 3. 太阳星（光明星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳火 |
| 斗系 | 中天星 |
| 化气 | 贵、官禄主 |
| 代表人物 | 比干 |

**性格特质：**
- **优点**：光明磊落、慷慨大方、有正义感、博爱无私
- **缺点**：过于直接、锋芒毕露、易招是非、劳心劳力

**关键特点：**
- 代表男性、父亲、上司
- 庙旺位置：寅卯辰巳午未（日出到日中）
- 落陷位置：申酉戌亥子丑（日落后）
- 白天生人太阳力量更强

**受亮度影响**：✅ 强烈受影响

**庙旺落陷：**
| 宫位 | 亮度 | 特点 |
|------|------|------|
| 寅卯辰 | 旺 | 朝气蓬勃 |
| 巳午 | 庙 | 光芒最盛 |
| 未申 | 得 | 开始西沉 |
| 酉戌亥 | 陷 | 失辉无力 |
| 子丑 | 陷 | 最暗无光 |

---

### 4. 武曲星（将星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴金 |
| 斗系 | 北斗 |
| 化气 | 财 |
| 代表人物 | 周武王 |

**性格特质：**
- **优点**：刚毅果断、理财有方、执行力强、重视效率
- **缺点**：孤僻固执、不善言辞、待人严苛、不近人情

**关键特点：**
- 财星，主正财、辛苦财
- 适合金融、财务、军警职业
- 刚克之星，需要柔和星曜调和

**受亮度影响**：❌ 不受影响

---

### 5. 天同星（福星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳水 |
| 斗系 | 南斗 |
| 化气 | 福、福德主 |
| 代表人物 | 周文王 |

**性格特质：**
- **优点**：心地善良、温和有礼、随和乐观、知足常乐
- **缺点**：缺乏斗志、做事拖延、不切实际、依赖性强

**关键特点：**
- 最祥和的福星
- 适合享受生活，不适合竞争环境
- 晚年运势通常较好

**受亮度影响**：✅ 受影响

---

### 6. 廉贞星（囚星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴火/阳木 |
| 斗系 | 北斗 |
| 化气 | 囚、官禄主 |
| 代表人物 | 费仲 |

**性格特质：**
- **优点**：负责尽职、积极进取、是非分明、敢做敢当
- **缺点**：心高气傲、情绪多变、锋芒太露、逞强好胜

**关键特点：**
- 古称"杀星"与"囚星"
- 十四主星中最高傲、最暴烈
- 事业主星，适合军警、政法
- 次桃花星

**受亮度影响**：✅ 受影响

## 二、天府星系（8颗）

### 7. 天府星（库星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳土 |
| 斗系 | 南斗主星 |
| 化气 | 贤能、田宅主 |
| 代表人物 | 姜太后 |

**性格特质：**
- **优点**：稳重保守、包容宽厚、理财有方、值得信赖
- **缺点**：过于保守、缺乏冲劲、不喜变化、安于现状

**关键特点：**
- 财库星，守财能力强
- 南斗之主，与紫微齐名
- 需看府库是否充盈（周围星曜配置）

**受亮度影响**：❌ 不受影响

---

### 8. 太阴星（月亮星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴水 |
| 斗系 | 中天星 |
| 化气 | 富、田宅主 |
| 代表人物 | 嫦娥 |

**性格特质：**
- **优点**：温柔细腻、富有内涵、艺术气质、善解人意
- **缺点**：多愁善感、优柔寡断、过于敏感、缺乏魄力

**关键特点：**
- 代表女性、母亲、妻子
- 财星，主不动产
- 夜晚生人太阴力量更强
- 庙旺：酉戌亥子丑（月升后）
- 落陷：卯辰巳午未（白天）

**受亮度影响**：✅ 强烈受影响

---

### 9. 贪狼星（桃花星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳木/阴水 |
| 斗系 | 北斗 |
| 化气 | 桃花、祸福 |
| 代表人物 | 妲己 |

**性格特质：**
- **优点**：多才多艺、交际广泛、机智灵活、欲望强烈
- **缺点**：贪多嚼不烂、好色贪财、投机取巧、不够专一

**关键特点：**
- **第一桃花星**
- 欲望之星，追求物质与情感
- 能文能武，适应性强
- 花钱大方，挥霍浪费

**受亮度影响**：✅ 受影响（庙旺身材好，落陷愚顽）

---

### 10. 巨门星（暗星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴水/阴土 |
| 斗系 | 北斗 |
| 化气 | 暗、是非 |
| 代表人物 | 马千金 |

**性格特质：**
- **优点**：口才犀利、分析力强、直言不讳、善于辩论
- **缺点**：多疑猜忌、好生是非、尖酸刻薄、人缘欠佳

**关键特点：**
- 是非之星，口舌之星
- 适合律师、教师、销售、主持
- 需要化禄化权才能转凶为吉

**受亮度影响**：✅ 受影响

---

### 11. 天相星（印星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳水 |
| 斗系 | 南斗 |
| 化气 | 印、官禄主 |
| 代表人物 | 闻仲太师 |

**性格特质：**
- **优点**：忠诚可靠、公正无私、善于协调、受人尊重
- **缺点**：过于保守、缺乏创新、容易受制、优柔寡断

**关键特点：**
- 宰相之星，辅佐之臣
- 需看"刑忌夹"或"财荫夹"
- 适合公务员、秘书、助理

**受亮度影响**：✅ 受影响（但影响较小）

---

### 12. 天梁星（荫星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阳土/阳木 |
| 斗系 | 南斗 |
| 化气 | 荫、寿 |
| 代表人物 | 李靖 |

**性格特质：**
- **优点**：正直善良、乐于助人、长者风范、化解灾厄
- **缺点**：好为人师、自以为是、管闲事、神经过敏

**关键特点：**
- 寿星、荫星
- 能化解灾厄，遇难呈祥
- 适合教育、医疗、公益

**受亮度影响**：✅ 受影响

### 13. 七杀星（将星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴金/阳火 |
| 斗系 | 南斗 |
| 化气 | 权、将星 |
| 代表人物 | 黄飞虎 |

**性格特质：**
- **优点**：勇敢果断、独立自主、有魄力、敢于挑战

**关键特点：**

- 喜紫微同宫或会照（化杀为权）

**受亮度影响**：❌ 不受影响

### 14. 破军星（耗星）

**基本属性：**
| 项目 | 内容 |
|------|------|
| 五行 | 阴水/阳水 |
| 斗系 | 北斗 |
| 化气 | 耗、夫妻主 |
| 代表人物 | 纣王 |

**性格特质：**
- **优点**：勇于开拓、敢于创新、不畏困难、破而后立

- 不利女命（缺乏小鸟依人气质）

**受亮度影响**：❌ 不受影响

## 三、星曜亮度系统

### 亮度等级

| 等级 | 含义 | 吉星表现 | 凶星表现 |
|------|------|---------|---------|
| 庙 | 最亮 | 极吉 | 不凶 |
| 旺 | 次亮 | 吉 | 不凶 |
| 得 | 明 | 仍吉 | 不凶 |
| 利 | 尚明 | 尚吉 | 渐凶 |
| 平 | 低 | 力微 | 肆凶 |
| 不 | 暗 | 无力 | 愈凶 |
| 陷 | 无光 | 无用 | 最凶 |

### 不受亮度影响的五颗星

```
紫微、天府、七杀、破军、武曲
```

这五颗星的特性相对固定，庙旺落陷影响较小。

### 强烈受亮度影响的星曜

```
太阳、太阴、天机、天同、巨门、贪狼、天相、廉贞、天梁
```
