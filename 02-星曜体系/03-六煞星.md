# 六煞星详解

## 概述

六煞星是紫微斗数中的凶星，会给命盘带来阻碍和挑战：

| 星曜 | 五行 | 安星依据 | 特性 |
|------|------|---------|------|
| 擎羊 | 阳金/阳火 | 年干 | 刑克、冲突 |
| 陀罗 | 阴金/阴火 | 年干 | 纠缠、拖延 |
| 火星 | 阳火 | 年支+时支 | 暴躁、冲动 |
| 铃星 | 阴火 | 年支+时支 | 阴沉、爆发 |
| 地空 | 阴火 | 时支 | 空亡、虚耗 |
| 地劫 | 阳火 | 时支 | 劫夺、断绝 |

---

## 一、擎羊陀罗（羊陀）

### 基本特性

| 项目 | 擎羊 | 陀罗 |
|------|------|------|
| 五行 | 阳金/阳火 | 阴金/阴火 |
| 别称 | 羊刃 | 马扫 |
| 位置 | 禄存前一宫 | 禄存后一宫 |
| 特性 | 明刀、直接冲突 | 暗箭、纠缠拖延 |

### 核心含义

- **擎羊**：如明刀在前，主刑克、冲突、伤害、竞争
- **陀罗**：如暗箭在后，主纠缠、拖延、反复、障碍

### 年干安星表

| 年干 | 禄存 | 擎羊 | 陀罗 |
|------|------|------|------|
| 甲 | 寅 | 卯 | 丑 |
| 乙 | 卯 | 辰 | 寅 |
| 丙 | 巳 | 午 | 辰 |
| 丁 | 午 | 未 | 巳 |
| 戊 | 巳 | 午 | 辰 |
| 己 | 午 | 未 | 巳 |
| 庚 | 申 | 酉 | 未 |
| 辛 | 酉 | 戌 | 申 |
| 壬 | 亥 | 子 | 戌 |
| 癸 | 子 | 丑 | 亥 |

### 庙旺落陷

| 宫位 | 亮度 | 说明 |
|------|------|------|
| 辰戌丑未 | 庙 | 四墓之地，煞气收敛 |
| 寅申巳亥 | 陷 | 四马之地 |
| 子午卯酉 | 陷 | 四正之地 |

### 重要格局

| 格局 | 含义 |
|------|------|
| 羊陀夹命 | 禄存在命宫，羊陀夹两侧，主孤独 |
| 羊陀夹忌 | 化忌星被羊陀夹，凶险 |
| 马头带剑 | 擎羊在午宫，可独当一面 |

### 擎羊特殊论法

- 擎羊在**子午卯酉**四正宫，注意灾伤
- 擎羊在**午宫**为"马头带剑"，反主武贵
- 擎羊庙旺加吉星，可化煞为权

## 二、火星铃星（火铃）

### 基本特性

| 项目 | 火星 | 铃星 |
|------|------|------|
| 五行 | 阳火 | 阴火 |
| 特性 | 明火、暴躁 | 阴火、闷烧 |
| 爆发 | 来得快去得快 | 积蓄后爆发 |

### 核心含义

- **火星**：性急暴躁、冲动、意外、血光
- **铃星**：阴沉内敛、突然爆发、隐藏危机

### 安星方法

火星铃星需要**年支**和**时支**共同确定。

**火星起点：**
| 年支 | 起点 |
|------|------|
| 寅午戌 | 丑宫起子时 |
| 申子辰 | 寅宫起子时 |
| 巳酉丑 | 卯宫起子时 |
| 亥卯未 | 酉宫起子时 |

**铃星起点：**
| 年支 | 起点 |
|------|------|
| 寅午戌 | 卯宫起子时 |
| 申子辰 | 戌宫起子时 |
| 巳酉丑 | 戌宫起子时 |
| 亥卯未 | 戌宫起子时 |

从起点**顺数**至出生时辰，即为火星/铃星位置。

### 庙旺落陷

| 年支 | 庙 | 落陷 | 得地 |
|------|----|----|------|
| 寅午戌 | 寅午戌 | 申子辰 | 亥卯未 |
| 其他 | 参照各派 | - | - |

### 火铃特殊组合

| 组合 | 含义 |
|------|------|
| 火铃夹命 | 性急躁动，易有灾厄 |
| 火贪格 | 火星+贪狼，暴发格局 |
| 铃贪格 | 铃星+贪狼，暴发格局 |
| 火铃羊陀 | 四煞聚会，大凶 |

### 火贪铃贪格

> **火贪格、铃贪格为横发格局**

火星或铃星与贪狼同宫或三方会照：
- 庙旺：暴发、意外之财
- 落陷：暴起暴落、昙花一现

## 三、地空地劫（空劫）

### 基本特性

| 项目 | 地空 | 地劫 |
|------|------|------|
| 五行 | 阴火 | 阳火 |
| 安星 | 亥宫起子时逆数 | 亥宫起子时顺数 |
| 特性 | 精神空虚 | 物质劫夺 |

### 核心含义

> **空劫又称"断桥煞"，主前路断绝、运势受阻**

- **地空**：精神层面的空虚，思想超脱，不切实际
- **地劫**：物质层面的损失，钱财劫夺，事业受阻

### 安星表

| 时辰 | 地空 | 地劫 |
|------|------|------|
| 子 | 亥 | 亥 |
| 丑 | 戌 | 子 |
| 寅 | 酉 | 丑 |
| 卯 | 申 | 寅 |
| 辰 | 未 | 卯 |
| 巳 | 午 | 辰 |
| 午 | 巳 | 巳 |
| 未 | 辰 | 午 |
| 申 | 卯 | 未 |
| 酉 | 寅 | 申 |
| 戌 | 丑 | 酉 |
| 亥 | 子 | 戌 |

### 空劫特点

| 特性 | 说明 |
|------|------|
| 主空亡 | 有形化无，得而复失 |
| 破财 | 钱财难聚，投资失利 |
| 宗教缘 | 适合哲学、宗教、玄学 |
| 创意 | 思维独特，有创造力 |

### 空劫特殊论法

- 空劫**本质是空亡**，庙旺落陷意义不大
- 空劫入命，适合从事**精神层面**工作
- 空劫与吉星同宫，**减半其吉**
- 空劫与凶星同宫，**增其凶**

### 空劫格局

| 格局 | 含义 |
|------|------|
| 命逢空劫 | 一生多波折，难聚财 |
| 财帛宫空劫 | 钱财难守，理财不当 |
| 空劫夹命 | 运势受阻，贵人难寻 |

## 四、四煞与六煞的关系

### 四煞

传统上"四煞"指：
```
擎羊、陀罗、火星、铃星
```

### 六煞

加上空劫二星：
```
擎羊、陀罗、火星、铃星、地空、地劫
```

### 煞星的双面性

煞星并非全然负面，在特定条件下可转化为助力：

| 条件 | 效果 |
|------|------|
| 煞星入庙 | 煞气收敛，反主刚毅 |
| 煞星化权 | 增强控制力 |
| 煞星与主星庙旺同宫 | 激发潜能 |
| 火贪/铃贪 | 横发格局 |
| 马头带剑 | 武贵格局 |

## 五、六煞星与命格

### 煞星入命的影响

| 煞星 | 入命影响 |
|------|---------|
| 擎羊 | 性格刚烈，易与人冲突，注意血光 |
| 陀罗 | 做事拖延，易有纠缠，反复不定 |
| 火星 | 性急暴躁，行事冲动，易有意外 |
| 铃星 | 内心压抑，突然爆发，隐藏危机 |
| 地空 | 思想超脱，不切实际，精神空虚 |
| 地劫 | 物质损失，钱财难聚，人生波折 |

### 煞星数量与格局

| 煞星数量 | 格局影响 |
|---------|---------|
| 1颗 | 可承受，增加人生历练 |
| 2颗 | 需注意，人生较多挑战 |
| 3颗以上 | 格局受损，需看主星强弱 |
| 四煞聚会 | 大凶，除非主星极强 |
