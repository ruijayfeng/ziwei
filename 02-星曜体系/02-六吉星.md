# 六吉星详解

## 概述

六吉星是紫微斗数中的辅助吉星，能增强命盘的格局：

| 星曜 | 五行 | 斗系 | 安星依据 | 成对出现效果更佳 |
|------|------|------|---------|-----------------|
| 左辅 | 阳土 | 北斗 | 月系 | 与右弼 |
| 右弼 | 阴土 | 北斗 | 月系 | 与左辅 |
| 文昌 | 阳金 | 南斗 | 时系 | 与文曲 |
| 文曲 | 阴水 | 北斗 | 时系 | 与文昌 |
| 天魁 | 阳火 | 南斗 | 年系 | 与天钺 |
| 天钺 | 阴火 | 南斗 | 年系 | 与天魁 |

---

## 一、左辅右弼（助力之星）

### 基本特性

| 项目 | 左辅 | 右弼 |
|------|------|------|
| 五行 | 阳土 | 阴土 |
| 斗系 | 北斗善星 | 北斗善星 |
| 安星 | 辰宫起正月顺数 | 戌宫起正月逆数 |
| 特性 | 阳刚、主动 | 阴柔、被动 |

### 核心含义

> **一字记之曰「助」**

- 代表**助力、贵人、辅佐**
- 增强主星的能量
- 提升格局层次

### 吉利表现

| 配置 | 效果 |
|------|------|
| 辅弼夹命 | 一呼百诺，贵人相助 |
| 辅弼同宫 | 丑未宫最佳，助力最大 |
| 辅弼拱照 | 三方四正相会，亦主助力 |
| 与紫微同宫 | 紫微得辅弼，群臣拥戴 |

### 同宫条件

左辅右弼同宫的月份：
- **四月生人**：未宫同宫
- **十月生人**：丑宫同宫

### 庙旺落陷

| 宫位 | 亮度 |
|------|------|
| 辰戌丑未 | 庙 |
| 其余宫位 | 旺 |

**注意**：左辅在戌酉宫力量较弱

---

## 二、文昌文曲（科甲之星）

### 基本特性

| 项目 | 文昌 | 文曲 |
|------|------|------|
| 五行 | 阳金 | 阴水 |
| 斗系 | 南斗 | 北斗 |
| 安星 | 戌宫起子时逆数 | 辰宫起子时顺数 |
| 特性 | 正统学术 | 才艺技能 |

### 核心含义

> **文昌主正统文才，文曲主才艺技能**

| 星曜 | 代表领域 |
|------|---------|
| 文昌 | 正统教育、科举、考试、文书、契约 |
| 文曲 | 艺术才华、音乐、表演、口才、异路功名 |

### 吉利表现

| 配置 | 效果 |
|------|------|
| 昌曲夹命 | 文采出众，金榜题名 |
| 昌曲同宫 | 丑宫最佳，未宫次之 |
| 与主星同宫 | 增添文雅气质 |
| 化科 | 名声、贵人、考运 |

### 同宫条件

文昌文曲同宫的时辰：
- **丑时生人**：未宫同宫（效果较弱）
- **未时生人**：丑宫同宫（效果最佳）

### 注意事项

- 昌曲虽为吉星，但**过于文弱**
- 不宜单独主事，需配合主星
- 文曲化忌主口舌是非、文书错误

## 三、天魁天钺（贵人之星）

### 基本特性

| 项目 | 天魁 | 天钺 |
|------|------|------|
| 五行 | 阳火 | 阴火 |
| 别称 | 天乙贵人/昼贵 | 玉堂贵人/夜贵 |
| 安星 | 年干 | 年干 |
| 适合 | 白天生人 | 夜晚生人 |

### 核心含义

> **魁钺主贵人、机遇、提携**

- 天魁为**阳贵人**：贵助明显，可直接感受
- 天钺为**阴贵人**：贵助暗中，沾光受惠

### 吉利表现

| 配置 | 效果 |
|------|------|
| 坐贵向贵 | 命宫魁、迁移宫钺（或反之） |
| 魁钺夹命 | 贵人相助，逢凶化吉 |
| 三方会照 | 亦主贵助 |
| 与主星同宫 | 增添贵气 |

### 年干安星表

| 年干 | 天魁 | 天钺 |
|------|------|------|
| 甲 | 丑 | 未 |
| 乙 | 子 | 申 |
| 丙 | 亥 | 酉 |
| 丁 | 亥 | 酉 |
| 戊 | 丑 | 未 |
| 己 | 子 | 申 |
| 庚 | 丑 | 未 |
| 辛 | 午 | 寅 |
| 壬 | 卯 | 巳 |
| 癸 | 卯 | 巳 |

### 重要特性

- 魁钺**不入辰戌二宫**
- 白天生人（卯辰巳午未申时）看天魁
- 夜晚生人（酉戌亥子丑寅时）看天钺

## 四、六吉星的层级效应

### 按安星依据分类

| 类别 | 星曜 | 影响周期 | 稳定性 |
|------|------|---------|--------|
| 年系星 | 天魁、天钺、禄存 | 终身 | 最稳定 |
| 月系星 | 左辅、右弼 | 大限 | 较稳定 |
| 时系星 | 文昌、文曲 | 流年 | 变化较快 |

### 六吉星优先级

在一般情况下的重要性排序：

```
1. 左辅右弼（助力最实际）
2. 天魁天钺（贵人机遇）
3. 文昌文曲（才华名声）
```

### 最佳配置

| 配置 | 格局名称 | 效果 |
|------|---------|------|
| 六吉星会聚命宫 | 六吉拱命 | 大富大贵 |
| 辅弼夹 + 魁钺夹 | 双夹格 | 贵人满门 |
| 昌曲夹 + 主星庙旺 | 文星拱命 | 文采斐然 |
