# 十二宫详解

## 概述

紫微斗数命盘由十二宫组成，从命宫开始**逆时针**排列：

```
命宫 → 兄弟 → 夫妻 → 子女 → 财帛 → 疾厄 →
迁移 → 交友 → 官禄 → 田宅 → 福德 → 父母
```

### 十二宫总览

| 宫位 | 主管领域 | 核心关键词 |
|------|---------|-----------|
| 命宫 | 自身命运 | 性格、才能、一生吉凶 |
| 兄弟宫 | 兄弟姐妹 | 手足、同事、合伙人 |
| 夫妻宫 | 婚姻感情 | 配偶、恋爱、婚姻质量 |
| 子女宫 | 子嗣关系 | 子女、下属、性生活 |
| 财帛宫 | 财富状况 | 赚钱能力、理财、财运 |
| 疾厄宫 | 健康疾病 | 身体、疾病、情绪 |
| 迁移宫 | 外出运势 | 社交、旅行、外在形象 |
| 交友宫 | 人际关系 | 朋友、下属、人缘 |
| 官禄宫 | 事业工作 | 职业、成就、社会地位 |
| 田宅宫 | 不动产 | 房产、家庭、财库 |
| 福德宫 | 精神生活 | 心态、兴趣、福报 |
| 父母宫 | 父母长辈 | 双亲、上司、遗传 |

---

## 一、命宫（核心宫位）

### 基本含义

> **命宫是命盘的核心，一切判断以命宫为基准**

| 项目 | 内容 |
|------|------|
| 主管 | 一生命运、性格、才能、思想、精神 |
| 代表 | 先天禀赋 + 后天发展 |
| 对宫 | 迁移宫 |
| 三方 | 财帛宫、官禄宫 |

### 看命宫要点

1. **主星**：决定基本性格和命格层次
2. **辅星**：增强或削弱主星能量
3. **四化**：动态变化的关键
4. **三方四正**：综合判断格局高低

### 命宫星曜配置

| 配置 | 影响 |
|------|------|
| 主星庙旺 + 吉星 | 格局高，一生顺遂 |
| 主星落陷 + 煞星 | 格局低，人生多波折 |
| 空宫借星 | 需看对宫星曜 |
| 煞星聚集 | 需主星强旺才能承受 |

---

## 二、兄弟宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 兄弟姐妹关系、数量、缘分 |
| 延伸 | 同事、同学、合伙人、手足情谊 |
| 对宫 | 交友宫 |
| 所属 | 兄疾田（财富资产三方） |

### 看兄弟宫要点

| 星曜 | 含义 |
|------|------|
| 吉星多 | 兄弟和睦、互相帮助 |
| 煞星多 | 兄弟不和、刑克 |
| 化忌 | 与兄弟有纠纷或缘薄 |
| 主星庙旺 | 兄弟有成就 |

### 现代延伸

- 母亲的情况（父母宫的夫妻宫）
- 工作合伙人关系
- 闺蜜/兄弟情

---

## 三、夫妻宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 婚姻、配偶、恋爱、感情 |
| 延伸 | 配偶性格、婚姻质量、离婚再婚 |
| 对宫 | 官禄宫 |
| 线位 | 夫官线（事业线/婚姻线） |

### 看夫妻宫要点

| 星曜 | 含义 |
|------|------|
| 紫微 | 配偶有地位、主导性强 |
| 天府 | 配偶稳重保守 |
| 贪狼 | 多桃花、感情复杂 |
| 巨门 | 口舌是非、沟通问题 |
| 煞星 | 婚姻波折、可能晚婚 |
| 化忌 | 感情纠葛、执念深 |

### 夫妻宫延伸

- 官禄宫的副业宫
- 福德宫的财帛宫（偏财位）
- 迁移宫的气数位

---

## 四、子女宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 子女缘分、数量、关系 |
| 延伸 | 下属、学生、晚辈、性生活 |
| 对宫 | 田宅宫 |
| 线位 | 子田线（老运、桃花） |

### 看子女宫要点

| 星曜 | 含义 |
|------|------|
| 吉星多 | 子女优秀、孝顺 |
| 煞星多 | 与子女缘薄、管教困难 |
| 空宫 | 可能少子或缘薄 |
| 化忌 | 为子女操心 |

### 子女宫延伸

- 桃花运势（合欢线）
- 下属能力和忠诚度
- 动产（车子等）

---

## 五、财帛宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 财富、理财、赚钱能力 |
| 延伸 | 金钱观念、财运起伏、投资 |
| 对宫 | 福德宫 |
| 三方 | 命宫、官禄宫 |

### 看财帛宫要点

| 星曜 | 含义 |
|------|------|
| 武曲 | 主财星，理财能力强 |
| 天府 | 守财能力强 |
| 太阴 | 不动产财、女性财 |
| 贪狼 | 偏财、投机财 |
| 破军 | 财来财去，难守 |
| 空劫 | 钱财难聚 |
| 化禄 | 财运旺盛 |
| 化忌 | 财运受阻 |

### 财帛宫与福德宫

财帛宫看**物质收获**，福德宫看**精神收获**，两宫互为对宫，需合参。

## 六、疾厄宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 健康、疾病、体质 |
| 延伸 | 情绪表达、灾厄、意外 |
| 对宫 | 父母宫 |
| 所属 | 兄疾田（财富资产三方） |

### 看疾厄宫要点

| 星曜 | 可能疾病 |
|------|---------|
| 太阳落陷 | 眼疾、心血管 |
| 太阴落陷 | 妇科、肾脏 |
| 廉贞 | 血光、心脏 |
| 天机 | 神经衰弱、四肢 |
| 巨门 | 口腔、消化道 |
| 火铃 | 烫伤、发炎 |
| 羊陀 | 外伤、手术 |

### 健康分析原则

- 需结合**父母宫**（遗传）
- 需结合**命宫**（整体状态）
- 运动可以补运

---

## 七、迁移宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 外出运、社交、外在表现 |
| 延伸 | 旅行、移民、人际关系、社会形象 |
| 对宫 | 命宫 |
| 影响 | 直接影响命宫格局 |

### 看迁移宫要点

| 星曜 | 含义 |
|------|------|
| 吉星多 | 外出发展有利、人缘好 |
| 煞星多 | 外出不利、注意意外 |
| 化禄 | 在外有财运 |
| 化忌 | 在外不顺 |

### 迁移宫重要性

> 迁移宫是命宫的对宫，其星曜吉凶**直接影响**命宫格局

- 迁移宫吉 → 命宫得助力
- 迁移宫凶 → 命宫受冲击

---

## 八、交友宫（奴仆宫）

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 朋友关系、交际能力 |
| 延伸 | 下属、员工、一般朋友 |
| 对宫 | 兄弟宫 |
| 古称 | 奴仆宫 |

### 看交友宫要点

| 星曜 | 含义 |
|------|------|
| 吉星多 | 朋友多、人缘好、得下属助力 |
| 煞星多 | 易被朋友连累、下属不力 |
| 化忌 | 与朋友有纠纷 |

### 与兄弟宫区别

- **兄弟宫**：血缘手足、深交好友
- **交友宫**：普通朋友、下属员工

---

## 九、官禄宫（事业宫）

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 事业、职业、工作、学业 |
| 延伸 | 社会地位、成就高低、职场运 |
| 对宫 | 夫妻宫 |
| 三方 | 命宫、财帛宫 |

### 看官禄宫要点

| 星曜 | 适合职业 |
|------|---------|
| 紫微 | 管理、政治、主导性工作 |
| 武曲 | 金融、财务、军警 |
| 天机 | 策划、幕僚、技术 |
| 太阳 | 公关、外交、光明正大行业 |
| 巨门 | 律师、教师、口才相关 |
| 廉贞 | 军警、政法、竞争性行业 |

### 官禄宫要素

1. **学习能力**：官禄宫首要看学习
2. **工作态度**：主动还是被动
3. **职业类型**：适合什么行业
4. **成就高低**：需与命财三方合看

| 项目 | 内容 |
|------|------|
| 主管 | 不动产、房屋、祖业 |
| 延伸 | 家庭环境、财库、家人相处 |
| 对宫 | 子女宫 |
| 所属 | 兄疾田（财富资产三方） |

### 看田宅宫要点

| 星曜 | 含义 |
|------|------|
| 天府 | 房产丰厚 |

| 武曲 | 可置产 |
| 破军 | 房产变动多 |
| 化忌 | 房产有纠纷 |

### 田宅宫延伸

- **财库**：钱财的储存能力
- **家运**：家庭是否和谐
- **居住环境**：房屋条件

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 精神生活、心态、福报 |
| 延伸 | 兴趣爱好、思想、享受能力 |
| 对宫 | 财帛宫 |
| 影响 | 潜意识、人生态度 |

### 看福德宫要点

| 星曜 | 含义 |
|------|------|
| 天同 | 知足常乐、享福 |
| 天梁 | 思想成熟、有福 |
| 贪狼 | 欲望多、贪图享乐 |
| 巨门 | 多思多虑、心烦 |
| 化忌 | 心中有执念 |

### 福德宫重要性

> 福德宫反映**内心世界**和**福报深浅**

- 福德宫吉 → 心态好、能享受人生
- 福德宫凶 → 精神压力大、难以放松

## 十二、父母宫

### 基本含义

| 项目 | 内容 |
|------|------|
| 主管 | 父母关系、长辈缘、遗传 |
| 延伸 | 上司老板、文书、政府 |
| 对宫 | 疾厄宫 |
| 影响 | 先天条件、童年环境 |

### 看父母宫要点

| 星曜 | 含义 |
|------|------|
| 吉星多 | 与父母关系好、得长辈助 |
| 煞星多 | 与父母缘薄、关系不佳 |
| 太阳 | 看父亲 |
| 太阴 | 看母亲 |
| 化忌 | 与父母有心结 |

## 十三、六线关系

### 三方四正

| 宫位 | 三方 | 四正（对宫） |
|------|------|-------------|
| 命宫 | 财帛、官禄 | 迁移 |
| 兄弟宫 | 疾厄、田宅 | 交友 |
| 夫妻宫 | 官禄、迁移 | 命宫（实为官禄对宫） |

### 六条宫线

| 线位 | 宫位 | 含义 |
|------|------|------|
| 命迁线 | 命宫-迁移宫 | 内外、动静 |
| 兄友线 | 兄弟-交友 | 人际关系 |
| 夫官线 | 夫妻-官禄 | 婚姻事业 |
| 子田线 | 子女-田宅 | 老运桃花 |
| 财福线 | 财帛-福德 | 物质精神 |
| 父疾线 | 父母-疾厄 | 遗传健康 |
