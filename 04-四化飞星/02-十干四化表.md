## 二、代码实现

```python
SIHUA_TABLE = {
    '甲': {'化禄': '廉贞', '化权': '破军', '化科': '武曲', '化忌': '太阳'},
    '乙': {'化禄': '天机', '化权': '天梁', '化科': '紫微', '化忌': '太阴'},
    '丙': {'化禄': '天同', '化权': '天机', '化科': '文昌', '化忌': '廉贞'},
    '丁': {'化禄': '太阴', '化权': '天同', '化科': '天机', '化忌': '巨门'},
    '戊': {'化禄': '贪狼', '化权': '太阴', '化科': '右弼', '化忌': '天机'},
    '己': {'化禄': '武曲', '化权': '贪狼', '化科': '天梁', '化忌': '文曲'},
    '庚': {'化禄': '太阳', '化权': '武曲', '化科': '太阴', '化忌': '天同'},
    '辛': {'化禄': '巨门', '化权': '太阳', '化科': '文曲', '化忌': '文昌'},
    '壬': {'化禄': '天梁', '化权': '紫微', '化科': '左辅', '化忌': '武曲'},
    '癸': {'化禄': '破军', '化权': '巨门', '化科': '太阴', '化忌': '贪狼'},
}

def get_sihua(tiangan):
    """获取指定天干的四化"""
    return SIHUA_TABLE[tiangan]

def get_sihua_star(tiangan, hua_type):
    """获取指定天干的某一化星"""
    return SIHUA_TABLE[tiangan][hua_type]

# 使用示例
print(get_sihua('甲'))
# 输出: {'化禄': '廉贞', '化权': '破军', '化科': '武曲', '化忌': '太阳'}

print(get_sihua_star('甲', '化禄'))
# 输出: 廉贞
```

## 四、重要规律

### 1. 高频化忌星

某些星曜更容易化忌，需特别注意：

| 星曜 | 化忌次数 | 天干 |
|------|---------|------|
| 天机 | 1 | 戊 |
| 太阳 | 1 | 甲 |
| 太阴 | 1 | 乙 |
| 武曲 | 1 | 壬 |
| 巨门 | 1 | 丁 |
| 天同 | 1 | 庚 |
| 廉贞 | 1 | 丙 |
| 贪狼 | 1 | 癸 |
| 文昌 | 1 | 辛 |
| 文曲 | 1 | 己 |

### 2. 双化星曜

某些星曜在同一天干中同时出现两种化：

| 天干 | 双化情况 |
|------|---------|
| 丁 | 天机化科、天同化权 |
| 庚 | 太阴化科、武曲化权 |
| 癸 | 太阴化科、巨门化权 |

### 3. 不参与四化的主星

以下主星**不参与四化**：
- 天府
- 天相
- 七杀

## 五、四化应用示例

### 示例1：生年四化分析

**甲年生人**：
- 廉贞化禄 → 事业心强，擅长管理
- 破军化权 → 有开创精神，敢于变革
- 武曲化科 → 理财有方，受人信任
- 太阳化忌 → 与男性长辈或上司关系需注意

### 示例2：大限四化分析

**行至戊干大限**：
- 贪狼化禄 → 该限桃花旺、欲望强
- 太阴化权 → 女性或房产方面有权力
- 右弼化科 → 有贵人相助
- 天机化忌 → 该限多变动、思虑过度
