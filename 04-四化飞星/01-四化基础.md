# 四化飞星基础

## 概述

四化是紫微斗数的核心动态分析系统，由天干引发星曜变化：

| 四化 | 名称 | 五行 | 含义 | 吉凶 |
|------|------|------|------|------|
| 化禄 | 禄 | 木/春 | 财富、福气、顺遂 | 吉 |
| 化权 | 权 | 火/夏 | 权力、压力、掌控 | 吉（带压力） |
| 化科 | 科 | 水/秋 | 名声、贵人、文采 | 吉 |
| 化忌 | 忌 | 金/冬 | 阻碍、纠缠、执念 | 凶 |

---

## 一、四化的本质

### 化禄
- **象征**：财禄、福泽、顺利
- **效果**：使星曜能量增强，带来物质与精神满足
- **特点**：得而欢喜，顺其自然

### 化权
- **象征**：权力、主导、竞争
- **效果**：增加星曜的控制力与影响力
- **特点**：争取而得，带有压力感

### 化科
- **象征**：名誉、贵人、学问
- **效果**：使星曜表现文雅、受人尊重
- **特点**：轻松得名，贵人相助

### 化忌
- **象征**：阻碍、执念、是非
- **效果**：使星曜能量受阻或过度执着
- **特点**：得而失，放不下

---

## 二、四化的来源层级

### 1. 生年四化（先天）

由**出生年干**决定，代表：
- 先天命格
- 业力牵引
- 一生基调

```
重要性：★★★★★
稳定性：终身不变
```

### 2. 宫干四化（后天）

由**各宫位天干**决定，代表：
- 宫位间的能量流动
- 自身努力方向
- 因果关系

```
重要性：★★★★☆
应用：飞星四化、自化分析
```

### 3. 大限四化（运势）

由**大限宫位天干**决定，代表：
- 十年运势趋向
- 阶段性机遇与挑战

```
重要性：★★★☆☆
周期：10年一变
```

### 4. 流年四化（时运）

由**流年天干**决定，代表：
- 当年运势特点
- 年度机遇与风险

```
重要性：★★☆☆☆
周期：1年一变
```

## 三、四化分析术语

### 化入与化出

| 术语 | 定义 | 效果 |
|------|------|------|
| 化入 | 四化飞入某宫 | 该宫获得能量 |
| 化出 | 从某宫飞出四化 | 能量外泄 |
| 自化 | 宫干化回本宫 | 得而复失 |

### 三吉化入规则

- **化入为实**：禄权科飞入的宫位，真正获益
- **化出为虚**：禄权科飞出的宫位，虚名虚利

### 自化详解

| 自化类型 | 含义 |
|---------|------|
| 自化禄 | 主动放弃，心甘情愿 |
| 自化权 | 争到最后仍失去 |
| 自化科 | 看破放下，心态平和 |
| 自化忌 | 失去后仍有麻烦 |

## 四、四化与星曜

### 可化星曜

并非所有星曜都能四化，只有以下星曜参与四化：

**十四主星（全部可化）：**
紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军

**辅星（部分可化）：**
文昌、文曲、左辅、右弼

### 四化分布规律

- 每个天干引发4颗星的变化
- 十个天干共引发40组四化
- 部分星曜出现频率较高（如武曲常化忌）

---

## 五、四化的宫位影响

### 命宫化禄

命宫星曜化禄 → 自身福气旺盛、财运佳

### 财帛宫化忌

财帛宫星曜化忌 → 财运受阻、理财需谨慎

### 三方四正的四化联动

四化不仅影响本宫，还通过三方四正传递：

```
命宫 ←→ 迁移宫（对宫）
命宫 ←→ 财帛宫 + 官禄宫（三合）
```

## 六、四化分析要点

### 1. 先看生年四化

确定命盘的先天格局：
- 三吉星（禄权科）落何宫？
- 化忌落何宫？

### 2. 再看宫干飞化

分析宫位间的能量关系：
- 命宫干化禄入财帛宫 → 自己努力赚钱
- 财帛宫干化禄入命宫 → 财来就我

### 3. 叠加运限四化

判断具体时间点的吉凶：
- 生年化忌 + 大限化忌同宫 → 该限运势低迷
- 生年化禄 + 流年化禄 → 该年财运极佳
